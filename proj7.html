Interactive Environment

Class: INFO C262 - Tangible User Interface

Role: Software Engineer

I designed an interactive environment based off of code from threejs . The idea was to use tangible inputs in the real world and map that to a virtual environment.

This project makes simple use of a potentiometer and a photoresistor in order to map motion and time of day to the environment. The potentiometer is used in order to turn perspectives int he environemnt. The photoresistor takes ambient light values and maps that to day and night.
