Dum-E: The Checkers Playing Robot

Class: ME 102B - Mechatronics

Role: Computer Vision and Software Lead

Dum-E is a checkers playing robot. It uses a webcam and a software package called openAI in order to register the position of the board state at set intervals. These board states are used as inputs to a handwritten checkers AI that uses alpha-beta pruning and a min-max algorithm to play against an opposing player.

As the software lead, I programed the AI as well as the computer vision code used. During our demo in Jacobs Hall at UC Berkeley in May 2016, Dum-E finished on top with a overwhelming score of 10-0 beating various students, staff, and visitors that stopped by.
